## Tips & Tricks

### Performance
- Use filters in Logs view to reduce data volume
- Pause auto-scroll when analyzing specific logs
- Clear old logs periodically with `c`

### Navigation
- Use number keys (1-4) for quick view switching
- `Home`/`End` keys for quick jumps in long lists
- `?` for contextual help anytime

### Monitoring
- Dashboard refreshes every 2 seconds automatically
- Watch health scores drop in real-time
- Monitor latency trends in charts

### Troubleshooting
- If view seems stuck, press `r` to refresh
- Check connection in Providers view with `t`
- Look for error patterns in Logs view
